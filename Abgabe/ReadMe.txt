To install the pictureCity Add-On make sure to have the Blender Version 3.4.0 installed.

Import the "install-prerequs.py" file into the Blender scripting workspace and run the file to install all necessary libraries in blenders python environment.

Then install the Add-On in the user preferences under 
"Edit->Preferences->Add-Ons->Install"
and then choosing the "picture-city-v1.0.0.zip" file. (Do not extract the .zip file yourself, Blender accepts the packed file itself)

The Add-On can now be found on the right side of the "3D Viewport in the "Toolbar" (Shortcut N) under the tab "PCity".
